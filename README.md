# Skills-Based Networking Assessment

## Objective
This Networking Assessment project aimed to design and configure a multi-VLAN network in Cisco Packet Tracer. The primary focus was on applying IPv4 and IPv6 addressing schemes, configuring routers and switches, implementing routing protocols, VLANs, trunking, EtherChannel, port security, STP, DHCPv4, SLAAC, and verifying network connectivity. This hands-on experience deepened understanding of network design, routing and switching concepts, and practical configuration skills.

### Skills Learned

- IPv4 subnetting using VLSM and IPv6 addressing implementation.  
- Configuration of routers, switches, VLANs, trunking, EtherChannel, and port security.  
- Implementation of static IPv4 and IPv6 routing, including redundant ISP links and backup routes.  
- Verification of connectivity using ping tests between VLANs, loopbacks, and simulated Internet.  
- Understanding of network topology design and documentation practices.  

### Tools Used

- [Cisco Packet Tracer](https://www.netacad.com/courses/packet-tracer) for network design and simulation.  
- Packet Tracer configuration files (`.pka`) to store the completed network.  
- Screenshots (`.png`) for visual representation of network topology.  
- PDF assessment requirements for reference.
